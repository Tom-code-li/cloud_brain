# 东软云医院 HIS 前端

## 本地运行

```bash
npm install
npm run dev
```

浏览器打开终端提示的地址，默认是 `http://127.0.0.1:5173`。

## 后端代理

- `/api` 默认代理到 `http://127.0.0.1:8080`
- `/medical-exam` 默认代理到 `http://127.0.0.1:9400`

如果检查、检验后端端口不同，请修改 `vite.config.js` 中的代理配置。
